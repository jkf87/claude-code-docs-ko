Week 15 · 2026년 4월 6–10일 ​

릴리스: v2.1.92 → v2.1.101, 신규 기능 4가지

Feature 1: Ultraplan (리서치 프리뷰) ​

터미널에서 클라우드 기반 Plan 모드를 시작하고, 브라우저에서 결과를 검토. Claude가 Claude Code on the web 세션에서 계획을 작성하는 동안 터미널은 자유롭게 사용 가능. 계획이 완성되면 각 섹션에 의견을 남기고, 수정을 요청하며, 원격 실행 또는 CLI로 되돌려받기를 선택할 수 있음. v2.1.101부터 첫 실행 시 기본 클라우드 환경이 자동 생성됨.

Feature 2: Monitor 툴 (v2.1.98) ​

백그라운드 워처를 실행하고 이벤트를 대화로 스트리밍하는 새로운 내장 툴. 각 이벤트가 트랜스크립트 메시지로 도착하면 Claude가 즉시 반응. 학습 실행 모니터링, PR CI 대기, 개발 서버 크래시 자동 복구 등에 활용 가능. /loop와 함께 사용하면 더욱 강력해짐 — 간격(interval)을 생략하면 Claude가 작업에 따라 다음 틱을 자동 스케줄링.

Feature 3: /autofix-pr (CLI) ​

PR 자동 수정은 Week 13에 웹에 먼저 도입됨. 이제 터미널을 떠나지 않고 활성화 가능. /autofix-pr이 현재 브랜치의 열린 PR을 자동 감지하여 Claude Code on the web에서 자동 수정을 활성화함.

Feature 4: /team-onboarding (v2.1.101) ​

로컬 Claude Code 사용 패턴을 기반으로 팀원 온보딩 가이드를 생성. 잘 아는 프로젝트에서 실행한 뒤, 결과물을 새 팀원에게 전달하면 됨.

기타 개선 사항 ​

  • 포커스 뷰: 깜빡임 없는 모드에서 Ctrl+O로 뷰 접기
  • Bedrock 및 Vertex AI 설정 마법사 가이드
  • /agents에 Running 탭이 포함된 탭형 레이아웃 적용
  • API 키 사용자의 기본 effort 레벨이 "high"로 변경
  • /cost에 모델별 및 캐시 적중률 상세 표시
  • /release-notes가 인터랙티브 버전 선택기로 개선
  • 상태 표시줄: refreshInterval 설정 지원
  • CLAUDE_CODE_PERFORCE_MODE 환경변수로 Perforce 지원
  • OS CA 인증서 저장소를 기본으로 신뢰
  • Amazon Bedrock이 Mantle 기반으로 구동
  • Bash 툴 권한 강화
  • UserPromptSubmit 훅에서 세션 제목 설정 가능
