# 새로운 기능

> Claude Code의 주요 기능을 매주 요약합니다. 코드 스니펫, 데모, 기능 배경 설명을 함께 제공합니다.

매주 개발 다이제스트에서는 업무 방식을 바꿀 가능성이 높은 기능을 중점적으로 소개합니다. 각 항목에는 실행 가능한 코드, 짧은 데모, 전체 문서 링크가 포함되어 있습니다. 버그 수정 및 사소한 개선 사항은 changelog를 참조하세요.

## Week 15 (2026년 4월 6–10일, v2.1.92–v2.1.101)

Ultraplan 클라우드 계획으로 프로젝트를 체계적으로 관리하고, Monitor tool로 실행 중인 프로세스를 실시간 모니터링할 수 있습니다. /autofix-pr CLI로 PR 리뷰 피드백을 자동 수정하고, /team-onboarding으로 팀원 온보딩을 간소화했습니다.

→ [Week 15 상세 보기](/whats-new/2026-w15)

## Week 14 (2026년 3월 30일–4월 3일, v2.1.86–v2.1.91)

CLI에서 Computer use를 직접 사용할 수 있게 되었고, /powerup 인터랙티브 레슨으로 핵심 기능을 단계별로 배울 수 있습니다. 깜빡임 없는 렌더링으로 출력 품질이 향상되었고, MCP result-size override와 Plugin executables on PATH로 확장성도 개선되었습니다.

→ [Week 14 상세 보기](/whats-new/2026-w14)

## Week 13 (2026년 3월 23–27일, v2.1.83–v2.1.85)

Auto mode로 Claude Code가 자율적으로 작업을 진행하고, Computer use (Desktop)로 데스크톱 환경을 직접 제어할 수 있습니다. PR auto-fix (Web)로 웹에서 리뷰 피드백을 자동 수정하고, Transcript search로 과거 대화를 검색하며, PowerShell tool과 Conditional hooks로 워크플로를 더 세밀하게 제어할 수 있습니다.

→ [Week 13 상세 보기](/whats-new/2026-w13)
