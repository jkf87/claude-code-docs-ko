# Week 14 · 2026년 3월 30일 – 4월 3일

릴리스: v2.1.86 → v2.1.91, 5개 신규 기능

## 기능 1: CLI에서 Computer use (리서치 프리뷰)

지난 주 Desktop 앱에 Computer use가 도입되었다면, 이번 주에는 CLI에서도 사용 가능. Claude가 네이티브 앱을 열고, UI를 클릭하고, 자신의 변경 사항을 테스트하고, 문제가 생기면 직접 수정 — 모두 터미널에서. 초기 단계이므로 불안정한 부분이 있을 수 있음.

## 기능 2: /powerup (v2.1.90)

터미널 내에서 애니메이션 데모를 통해 Claude Code 기능을 가르치는 인터랙티브 레슨.

## 기능 3: 깜빡임 없는 렌더링 (v2.1.89)

가상화된 스크롤백을 갖춘 새로운 alt-screen 렌더러를 선택적으로 사용. `CLAUDE_CODE_NO_FLICKER=1`로 설정.

## 기능 4: MCP 결과 크기 오버라이드 (v2.1.91)

MCP 서버 작성자가 `tools/list`에 `"anthropic/maxResultSizeChars"`를 설정하여 특정 툴의 잘림 상한을 최대 50만 자까지 올릴 수 있음.

## 기능 5: 플러그인 실행 파일 PATH 추가 (v2.1.91)

플러그인 루트의 `bin/` 디렉터리에 실행 파일을 넣으면 Claude Code가 Bash 툴의 PATH에 해당 디렉터리를 자동 추가.

## 기타 개선 사항

- Auto mode 후속 처리: 분류기 거부 시 PermissionDenied 훅 지원
- PreToolUse 훅의 `permissionDecision`에 새로운 `"defer"` 값 추가
- `/buddy`: 코딩하는 모습을 지켜보는 작은 생명체 부화 (4월 1일)
- `disableSkillShellExecution` 설정 추가
- Edit 툴이 `cat` 또는 `sed -n`으로 본 파일에도 동작
- 50K 이상의 훅 출력은 디스크에 저장
- Thinking 요약 기본 해제
- 음성 모드: push-to-talk 수정자 조합 지원
- `claude-cli://` 딥 링크에서 여러 줄 프롬프트 지원
